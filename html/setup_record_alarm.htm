If you want to set the time interval, drag the mouse to the time table at the top of the screen and a recording information setting window should appear.

In the recording information settings window, you can set the desired resolution and quality, FPS, audio recording, and whether to save or not.

If there is a different setting value, the time table will be displayed in a different color.

If you hover the mouse pointer over the top of the time table, the corresponding time zone information will be displayed for the recording settings.

Change the mode of schedule in operation mode page to change hourly, daily setup.
